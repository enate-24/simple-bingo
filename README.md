# simple-bingo

Professional Lottery Draw System built with React, TypeScript, and Tailwind CSS.

## Features

- Interactive number selection (1-20)
- Random winner drawing with animations
- Sound effects on draw
- Professional UI with gradient designs
- Responsive layout for all devices
- Progress tracking and visual feedback

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Lucide React Icons

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Usage

1. Select numbers from the grid (1-20)
2. Click "Draw Winner" to randomly select from your chosen numbers
3. Winners are displayed with animations
4. Reset to start a new draw
